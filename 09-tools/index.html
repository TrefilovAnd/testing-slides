ESlint Установка


# Установка eslint

ESlint Конфигурация


// .eslintrc.json

{
    "env": {
        "es6": true,
        "node": true
    },
    "parser": "babel-eslint",
    "rules": { /* ... */}
}
        

./node_modules/.bin/eslint --init
